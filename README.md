# English Words to Anki Deck Generator ????????

Скрипт для генерации карточек Anki из списка английских слов.  
Для каждого слова автоматически добавляются:

- ?? Транскрипция (с помощью `phonemizer`)
- ?? Озвучка (с помощью `gTTS`)
- ?? Перевод на русский (с Google Translate)
- ?? Пример предложения с переводом (с сайта Tatoeba)
- ??? Картинка (с Pexels API)
- ?? Всё экспортируется в `anki_deck.csv` (совместимо с Anki)

---

## ?? Структура проекта

```text
+-- main.py             # Основной скрипт
+-- words.txt           # Список слов (по одному на строку)
+-- audio/              # Папка с mp3-файлами
+-- images/             # Папка с изображениями
+-- anki_deck.csv       # Результирующий CSV для Anki
+-- .env                # Файл с ключом Pexels API
+-- .gitignore
+-- requirements.txt

?? Установка
1. Установите Python 3.8+
2. Установите зависимости:

pip install -r requirements.txt

Установите eSpeak NG и проверьте, что он установлен в:
C:\Program Files\eSpeak NG

Создайте файл .env в корне проекта и добавьте в него ваш Pexels API ключ:
PEXELSAPIKEY=вашключотpexels

Создайте файл words.txt и впишите слова (по одному в строку):
apple
house
smile


?? Запуск

python anki_cards.py
По завершении в файле anki_deck.csv будут собраны данные для импорта в Anki.

??? Примечания
?? Убедитесь, что eSpeak NG установлен, иначе phonemizer не сработает.

Скрипт делает запросы к внешним сервисам, поэтому лучше запускать с паузой (time.sleep(1)) - чтобы избежать блокировок.

Для импорта в Anki используйте CSV с разделителями , и кодировкой UTF-8.
